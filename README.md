Zomato Restaurant Analytics Dashboard

📊 Project Overview

This project analyzes Zomato restaurant data using Excel to find insights on ratings, votes, cuisines, and city performance.

📸 Dashboard Preview

![Dashboard](Screenshot 2026-04-03 151531.png)

🔍 Key Insights

- Top cuisines based on popularity
- City-wise restaurant performance
- Highest rated restaurants
- Voting trends

🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

📁 Dataset

Zomato dataset

🚀 How to Use

Download the Excel file and interact with slicers.
